# Portafolio Bootstrap

Este proyecto es un portafolio personal construido con HTML, CSS y Bootstrap.

## Estructura de archivos

- index.html - página principal del portafolio
- 
osotros.html - página sobre mí y formación
- proyectos.html - página de proyectos destacados
- contacto.html - página de contacto y formulario- `servicio-soporte.html` - página especial para el servicio de soporte técnico
- `servicio-desarrollo.html` - página especial para el servicio de desarrollo web
- `pagina-ejemplo.html` - plantilla de página adicional- css/styles.css - estilos personalizados del sitio
- js/main.js - lógica mínima para actualizar el año en el pie de página
- components/navbar.html - componente de navegación (para uso futuro)
- components/footer.html - componente de pie de página (para uso futuro)

## Uso

1. Abre index.html en tu navegador.
2. Lo ideal es abrirlo desde un servidor local como Live Server, porque `fetch()` carga los componentes desde `components/`.

## Notas

- Bootstrap se carga desde CDN.
- Los componentes `navbar` y `footer` se cargan con JavaScript para mantener una estructura centralizada.
- Reemplaza los textos y datos de contacto con tu información real.
